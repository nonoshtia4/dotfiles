# Nono's Dotfiles

## Contents

- vim (NeoVim) config
  - Plugins are managed with [dein.vim](https://github.com/Shougo/dein.vim)
- tmux config
- git config

## How to use

- init.sh
- If you need to clear dein cache
- :call dein#recache_runtimepath()
